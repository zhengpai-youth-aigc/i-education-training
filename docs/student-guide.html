零基础，用 Codex
做出你的第一个网页

不需要会写代码，不需要懂技术。只要你会说话，就能用 Codex 从 0 做一个小工具，帮你整理工作任务。

→ 就像这篇文章里写的那样，我带你一步一步做

📦 你会做出这样的东西

输入你是做什么的、你有什么任务、想要什么结果 → 点一下 → 自动生成任务拆解、可复制的提示词、执行步骤清单。

0课前准备

先把这三样东西准备好，做的时候不用分心

✓

一个 ChatGPT 账号
Codex 需要登录 ChatGPT 账号才能用。官网注册免费：chatgpt.com

✓

VS Code 编辑器
免费，Mac 和 Windows 都能装。官网：code.visualstudio.com

✓

Codex 扩展
VS Code 装好后，点左侧扩展图标，搜索 "Codex"，找到那个蓝色的插件，点安装。

1打开 Codex

VS Code 装好了，接下来打开它

①

VS Code 左侧边栏找 Codex 图标

安装完扩展后，VS Code 左侧边栏会出现一个 Codex 的图标（一个聊天气泡里面有个闪电）。点击它。

②

用 ChatGPT 账号登录

第一次点开 Codex，它会让你登录。用你的 ChatGPT 账号登录就可以了（推荐 Plus 账号，免费账号也能用，但速度可能稍慢）。

③

新建一个文件夹

在 VS Code 里点 File → New Folder（或者在桌面新建一个文件夹），起名叫"我的工作助手"。然后 VS Code 里点 File → Open Folder，打开这个文件夹。

2告诉 Codex 你要做什么

这是最关键的一步——把你的想法"翻译"成 Codex 能听懂的话

📋

直接复制粘贴这段话

打开 Codex 的对话框，把下面这段话直接粘贴进去：

我想做一个"工作任务助手"网页。

功能：输入我的职业身份（比如"我是外贸业务员"）、当前的工作任务、希望产出的结果形式，页面自动整理出：
1. 任务拆解（分步骤）
2. 可复制的提示词（可以直接发给 AI 用）
3. 执行步骤清单

界面要简洁好看，用暖色调配色，像笔记本的感觉。

请从 0 开始帮我做。

↑ 直接复制，Ctrl+C / Cmd+C，然后粘贴进 Codex 对话框

▶

发送，等 Codex 回应

粘贴好后，按回车发送。Codex 会开始"思考"——它可能会问你一些问题，也可能会直接开始帮你做文件。等它一下，通常几秒钟到几十秒。

💡 如果等了很久显示"正在思考..."，按 Ctrl+C 可以让它停下来，然后重新发送一次。

3跟着 Codex 一步一步做

Codex 会告诉你它要做什么、创建什么文件、修改什么代码

👍

它说什么你就做什么

Codex 可能会说"我帮你创建了 index.html 文件，现在帮你添加样式..."这类话。它说完一段，你就等它做完，然后看它下一步说什么。

✏️

不满意就说出来

如果做完的东西你觉得颜色不对、布局不对、功能不对，直接说！比如"把背景色改成蓝色"、"把字体变大一点"——Codex 会帮你改。

🔄

做完了让它演示一下

全部做完之后，告诉 Codex："帮我用浏览器打开看看"。它会告诉你怎么在浏览器里打开预览。

💡 或者直接在 VS Code 里右键点击生成的 HTML 文件，选择 "Open with Live Server"，浏览器会自动打开。

4测试你的成果

现在你有一个真实可用的网页了——测一下

🧪

输入你真实的工作

在网页里输入你是做什么的（比如"我是做电商运营的"），输入你最近在做的任务（比如"要写一份产品上架文案"），选择输出形式，看它帮你整理出什么。

🎯

改到满意为止

觉得哪里不对，回到 Codex 说"我想加一个功能：导出成文字"，或者"我觉得这个颜色不好看，换成绿色"——它会帮你改，改完再刷新浏览器看。

常见问题

Q: 等了很久 Codex 一直"正在思考"，怎么办？

A: 按 Ctrl+C（Mac 是 Ctrl+C）可以终止。然后重新发送一次需求，或者把需求说得更简短一些。

Q: 生成的页面样式不对，怎么改？

A: 直接在 Codex 里说清楚你想怎么改，比如"把字体改成 18px"、"把按钮颜色改成红色"，它会帮你调整 HTML 或 CSS。

Q: 完全没有编程基础，能做吗？

A: 可以。这篇文章的作者也是从零开始折腾的，Codex 就是为了零基础的人设计的。你只需要会说话、会复制粘贴，就能做。

Q: Windows 用户也能做吗？

A: 能。VS Code 和 Codex 扩展 Windows 和 Mac 都有，操作步骤一模一样。
