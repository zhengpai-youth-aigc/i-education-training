第一课 · 零基础入门

Codex 入门：装好、打开、体验一下

跟着做，30分钟完成。不需要会写代码，不需要懂技术。

一、Codex 是什么

先做个小小的科普，因为很多朋友可能完全没听过这个东西。

相信很多朋友都知道 Claude Code 和大龙虾 Clawdbot 对吧？简单说，这俩都是一个叫 Agent 的东西——你可以理解为一种特别能干的 AI 助手，它不只能聊天，还能帮你真正动手做事情，比如写代码、改文件、跑程序。

Codex，就是 OpenAI 出的 Claude Code，对标 Anthropic 家的产品。都是编程 Agent，都能帮你真正把事情做成。

顺便说一下，Codex 和 Claude Code 都是工具，它们还需要背后的"大脑"——也就是模型。比如 GPT-5.3-codex 是 OpenAI 专门为编程优化的模型，在创作写作、查资料这些方面反而不太行，所以 OpenAI 没有把它放到普通 ChatGPT 里给大家用。

二、Codex 能干什么

🖥️ 做网页和小工具

告诉它你想做什么样的网页，它直接帮你写好代码、跑起来。不需要会 HTML、CSS、JavaScript。

📄 写各种文档和文案

商业邮件、方案文档、工作总结——告诉它你的需求，它帮你写出专业的中文文案。

📊 处理数据和分析

批量重命名文件、整理表格、统计数据——那些费时费力的重复性工作，告诉 Codex，它帮你做完。

🔍 读懂代码、解释内容

把一段代码、一篇文章丢给 Codex，让它帮你解释、分析、总结要点。用你能看懂的话讲给你听。

和 ChatGPT 有什么区别？

ChatGPT — 军师，给你建议和答案

你问它答，适合聊天、写作、分析。但最终动手的事情——写代码、生成文件——还得你自己来。

Codex — 助理，帮你把事情做完

你说需求，它直接帮你做出来。生成代码、运行程序、生成文件——你看到的是可以直接用的结果。

本质区别：ChatGPT 告诉你答案，Codex 帮你把事情做成。

三、下载与安装

⚠️ 国内用户需要先准备 VPN（必须用美国节点）
  • openai.com 和 Mac App Store 在国内打不开，需要 VPN 才能访问
  • 必须用美国节点（其他地区节点可能无法注册/登录 OpenAI）
  • Mac 用户还需要一个美国区 Apple ID才能在 App Store 下载
  • 便宜的 VPN 推荐：点击购买（够便宜，买了挂美国节点就能用）
安装步骤
1
开 VPN，确保能访问 openai.com
2
选一种方式下载安装（见下方）
3
打开 App，用 ChatGPT 账号登录
4
开始用中文描述你的需求，Codex 帮你完成！
没有账号？去 chatgpt.com 注册（免费），或在闲鱼买 Plus 账号合租（约10-15元/月）。
方式一：桌面客户端（新手推荐 ⭐）
🍎 Mac 用户

像普通软件一样安装，点开就能用，有图形界面，操作最简单。

🌐 官网下载 🍎 Mac App Store
🪟 Windows 用户

Codex 已推出 Windows 原生客户端，下载安装后登录即可使用，和 Mac 版体验一致。

🌐 官网下载 🐙 GitHub
方式二：VS Code 编辑器插件

如果你已经在用 VS Code，可以直接装 Codex 插件，在编辑器里用，不需要下载额外软件。

🔌 在你的 IDE 中试用
方式三：命令行（适合有基础的）

打开终端，输入以下命令安装：

Mac 用户：

➜ ~ brew install --cask openai-codex
➜ ~ open -a Codex

Windows 用户先运行 wsl --install 装好 WSL 再执行：

➜ ~ npm install -g @openai/codex
added 1 package in 2.3s
➜ ~ codex
✓ Codex 已启动
