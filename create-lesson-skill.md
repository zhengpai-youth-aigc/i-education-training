# 创建课程skill

## 需求说明

将输入的内容经过处理后，按照项目规范将其转换为合适的页面，目前项目对应的页面有 课程 和 资源
其中资源中的学习资源页面是一些推荐账号（公众号或B站账号）。 这些页面类型后续项目迭代可能会变更，因此需要以最新项目的规范为准

因此需要基于用户的输入来进行意图判断，用户是需要将其输入生成一个课程页还是资源页，还是说只是将其添加到学习资源的推荐中
如果用户输入需要中明确提及可以优先以用户需求为准，但主要需要Claude code基于上下文与当前项目的最新规范结合判断，这些输入内容最好放置在什么地方(给出推荐意见让用户进行选择)

用户的输入可能包括本地文件夹，网络链接等等。
对输入进行检索等操作的时候，需要优先使用本地配置好的mcp或skill。例如输入中包含了网络链接，先检查当前调用者本地是否存在相关用于搜索的mcp工具或者skill（例如grok-search），优先使用mcp等工具处理

然后根据本地最新的规范（也就是trellis的规范文件）进行页面修改的处理
